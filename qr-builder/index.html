QR Builder (canvas)

You can hover the image to view the computed text from the "title" property.


Some things to try out.

  • (default) Homer J. Simpson's contact info (vcard format)
  • (url) https://wikipedia.org
  • (phone call) tel:+1234567890
  • (text message) smsto:+1234567890:hello there!
  • (wifi network info) WIFI:T:WPA;S:somenetworkname;P:myamazingpassword;;

Alternatively...

You could pass a value to this page as the query parameter 'value'. For example:
https://edm00se.codes/demos/qr-builder/?value=Hello World!
